# XC 语言语法规范 V4（语法2 - 简化版）

## 设计原则

- 简洁：一个符号代表一个含义
- 形意合一：形状暗示功能
- 接近 C 结构：易于转换为 C/Rust/Mojo

## 5. 控制流

### 5.1 if 语句

```
? (x > 0) {
    // 代码
}
```

### 5.2 if-else 语句

```
? (x > 0) {
    // x > 0 时执行
} ?: {
    // x <= 0 时执行
}
```

### 5.3 if-elif-else 语句

```
? (x > 0) {
    ! "positive"
} ?? (x < 0) {
    ! "negative"
} ?: {
    ! "zero"
}
```

### 5.4 while 循环

```
@(i < 10) {
    ! i
    i = i + 1
}
```

### 5.5 for 循环

```
~i = 0; i < 10; i = i + 1 {
    ! i
}
```

格式：`~初始化; 条件; 步进 { 循环体 }`

### 5.6 break 和 continue

```
~i = 0; i < 10; i = i + 1 {
    ? (i == 5) {
        >      // break，跳出循环
    }
    ? (i == 3) {
        <      // continue，跳过本次迭代
    }
    ! i
}
```

## 14. 示例程序

### Hello World

```
# {
    ! "Hello, World!"
}
```

### 累加求和

```
# {
    $sum = 0
    ~i = 1; i <= 100; i = i + 1 {
        sum = sum + i
    }
    ! "Sum = ", sum
}
```

### 阶乘函数

```
% fac(n: int) -> int {
    ? (n <= 1) {
        ^ 1
    } ?: {
        ^ n * fac(n - 1)
    }
}

# {
    ! fac(5)
}
```

### 二分查找

```
% binary_search(arr: int[], n: int, target: int) -> int {
    $left = 0
    $right = n - 1

    @ (left <= right) {
        $mid = (left + right) / 2
        ? (arr[mid] == target) {
            ^ mid
        } ?? (arr[mid] < target) {
            left = mid + 1
        } ?: {
            right = mid - 1
        }
    }
    ^ -1
}

# {
    $arr = [1, 3, 5, 7, 9, 11]
    ! binary_search(arr, 6, 7)
}
```

## 15. 与 C 的语法对照

| XC | C |
|----|---|
| `# { }` | `int main() { }` |
| `$x = 10` | `int x = 10;` |
| `@PI = 3.14` | `const double PI = 3.14;` |
| `% f(a,b) -> int { }` | `int f(int a, int b) { }` |
| `^ expr` | `return expr;` |
| `? (cond) { }` | `if (cond) { }` |
| `?: { }` | `else { }` |
| `?? (cond) { }` | `else if (cond) { }` |
| `@ (cond) { }` | `while (cond) { }` |
| `~i=0; i<10; i=i+1 { }` | `for (int i=0; i<10; i++) { }` |
| `> ` | `break;` |
| `< ` | `continue;` |
| `! x` | `printf("%d\n", x);` |
| `?I($x)` | `scanf("%d", &x);` |
| `& Point { }` | `struct Point { }` |
| `&Point{ x: 1, y: 2 }` | `struct Point p = {1, 2};` |
| `x as int` | `(int)x` |

## 17. 语法糖

### 17.1 复合赋值

```
x += 1      // 等同于 x = x + 1
x -= 1      // 等同于 x = x - 1
x *= 2      // 等同于 x = x * 2
x /= 2      // 等同于 x = x / 2
```

### 17.2 自增自减

```
x++         // 等同于 x = x + 1
x--         // 等同于 x = x - 1
++x         // 同上（前缀）
--x         // 同上（前缀）
```

### 17.3 三元运算符（可选）

```
$x = (a > b) ? a : b
```

## 18. 错误处理（可选扩展）

```
% safe_div(a: int, b: int) -> Result<int, string> {
    ? (b == 0) {
        ^ Err("division by zero")
    } ?: {
        ^ Ok(a / b)
    }
}
```
